# Игра «Найди свой ОКВЭД по номеру телефона»

Консольная игра на Java, которая по российскому мобильному номеру определяет «твой» вид деятельности по классификатору ОКВЭД 2 — ищет код с **максимальным совпадающим суффиксом** в конце номера.

## Как запустить (один файл — всё, что нужно знать)

### 1. Требования

- Java 11 или новее  
- Maven (рекомендуется) или Gradle  
- Доступ в интернет (для первой загрузки справочника okved.json)

### 2. Склонировать репозиторий

```bash
git clone (https://github.com/Ionina34/Okved-Game.git)
cd Okved-Game
```

### 3. Собрать проект
## Maven (рекомендуемый способ)
mvn clean package

## или Gradle (если используешь)
./gradlew build

### 4. Запустить игру
```bash
java -jar target/okved-game-1.0-SNAPSHOT.jar
```

### Зависимости
Проект использует только одну внешнюю библиотеку:
Gson — для парсинга JSON-файла okved.json

Ссылка: https://github.com/google/gson
